## 환경 설정
     XAMPP
       - Apache와 MySQL 사용
       - port 번호 다음과 같이 설정
     phpMyAdmin
       - MySQL 옆의 Admin을 눌러서 phpMyAdmin에 접속
     
     - 프로젝트 파일의 경로는 D:\xampp\htdocs\myproject
     - http://localhost/myproject/index.php URL로 접속
